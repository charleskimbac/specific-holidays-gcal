Add Specific Holidays to Google Calendar

 GitHub

Directions

1) Go to Google Calendar (website) settings > "Add calendar" > "Browse calendars of interest" > "Regional holidays" > Browse and select a calendar.

2) Select the added calendar under "Settings for other calendars" > "Integrate calendar" > Copy the entire link in "Public address in iCal format"

3) Open a new tab and visit the link. A .ics file should download.

4) Upload and submit the file below. Then, submit the specific holidays, copy the output at the bottom of the page, and paste it in the script (after making a copy) as the value for SPECIFIC_HOLIDAYS.


FAQ

1) What does "(substitute)" mean? If the holiday falls on the weekend, the succeeding weekday (usually Monday) is designated as the holiday "substitute".

2) "Election Day" vs. "Election Day (General Election)"? No clue, just use "Election Day (General Election)".
